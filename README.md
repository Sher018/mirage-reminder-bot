# Mirage Reminder Bot

Бот-напоминалка для Ресторанного комплекса Мираж, Иркутск.

## Быстрый старт

1. **Установите зависимости:**
   ```
   pip install -r requirements.txt
   ```

2. **Настройте .env** — добавьте свой ID в `ADMIN_IDS` (узнать: @userinfobot)

3. **Установите зависимости** (если ещё не установлены):
   ```
   pip install -r requirements.txt
   ```
   или
   ```
   py -m pip install -r requirements.txt
   ```

4. **Запустите бота:**
   ```
   py main.py
   ```

5. **Проверьте** — напишите боту `/start` в Telegram

Команды: `/schedule` или `/raspisanie` — загрузить расписание.

## Остановка бота

В терминале нажмите **Ctrl+C**.

## Использование

1. **Добавьте бота в группу** с работниками
2. **В группе** выполните `/setgroup` — сохранить группу для напоминаний
3. **В личку боту** отправьте `/schedule` и затем Excel-файл (дата, день, username, начало, конец)

Напоминания будут приходить в группу по времени начала смен. Работники подтверждают геолокацией (обязательно) и фото (по желанию). Воскресенье 00:00 — отчёт админу в личку.

---

## Важно: безопасность токена

⚠️ **Токен бота скомпрометирован** (был показан в чате). Рекомендуется:
1. Открыть @BotFather в Telegram
2. Отправить `/revoke`
3. Выбрать своего бота
4. Получить новый токен
5. Обновить `BOT_TOKEN` в файле `.env`
