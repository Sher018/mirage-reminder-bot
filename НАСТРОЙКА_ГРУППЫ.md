# Настройка группы и подтверждений

## 0. Group Privacy — обязательно выключить!

В настройках бота в @BotFather:
- **Group Privacy** → **Turn OFF** (выключить)

Если включено — бот **не получает** фото и геолокацию от работников. Напоминания отправлять может, но подтверждения не увидит.

## 1. Сохранить группу для напоминаний

**Вариант А (рекомендуется):**
1. Добавьте бота в группу с работниками
2. В группе выполните команду **`/setgroup`**
3. Бот сохранит ID группы

**Вариант Б:**
1. В группе выполните **`/groupid`** — бот покажет ID
2. Добавьте в файл `.env`: `GROUP_ID=-1001234567890` (ваш ID)

## 2. Отключить Privacy Mode (обязательно!)

Чтобы бот **видел фото и геолокацию** от работников:

1. Откройте **@BotFather** в Telegram
2. Отправьте **`/mybots`**
3. Выберите своего бота
4. **Bot Settings** → **Group Privacy** → **Turn off**

Иначе бот не получит сообщения без упоминания (@bot).

## 3. Проверка

- **`/check`** — проверка настроек (в личку боту)
- **`/test`** — отправить тестовое сообщение в группу

## 4. Координаты ресторана

В `.env` укажите точные координаты:
- `RESTAURANT_LAT` — широта
- `RESTAURANT_LON` — долгота
- `GEO_RADIUS_M` — радиус в метрах (по умолчанию 200)

Геолокация работника должна быть в этом радиусе.
