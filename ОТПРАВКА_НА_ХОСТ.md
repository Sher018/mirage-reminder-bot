# Загрузка изменений на хост (GitHub → Amvera)

Выполните в **PowerShell** или **Терминал** в папке проекта:

```powershell
cd c:\RUN-RAY

# 1. Посмотреть, что изменилось
git status

# 2. Добавить все изменения
git add .

# 3. Создать коммит с описанием
git commit -m "Напоминания за 5 мин до смены: полный день, без уже отметившихся в прошлых сменах; /test_reminder 12:00 для теста"

# 4. Отправить на GitHub
git push origin main
```

Если ветка называется не `main`, а например `master`, замените на:

```powershell
git push origin master
```

---

**После push:**

- Если Amvera подключён к репозиторию GitHub, деплой может запуститься автоматически.
- Иначе откройте проект на [cloud.amvera.ru](https://cloud.amvera.ru), вкладка «Деплой» или «Сборка», и нажмите **«Пересобрать»** / **«Redeploy»**.

Готово: изменения будут на хосте.
